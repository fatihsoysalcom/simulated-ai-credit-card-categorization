# Simulated AI Credit Card Categorization

This example demonstrates the core concept of using AI to categorize credit card transactions. It simulates an AI model's ability to process transaction descriptions and assign them to predefined financial categories, helping users understand their spending habits automatically. The `categorize_transaction` function mimics how an LLM like Gemma 4 would analyze text and classify it.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Run from your terminal: `python main.py`
